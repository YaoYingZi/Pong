# Pong
对电子游戏Pong的重现，~~仅支持人机对战~~，使用虚幻引擎5.7.4的蓝图制作。

A recreation of Pong, one of the earliest and most influential video games. ~~This version only supports PVE~~ and is developed entirely with Unreal Engine 5.7.4 Blueprint scripting.

## 2026/7/18 重大更新：
### 目前游戏已支持单机PVP与PVE
### 具体更新内容：
* 优化了球体发射方向的生成逻辑
* 优化了球体生成时刻与生成间隔
* 对部分蓝图的重构与逻辑优化
* 添加单机PVP模式

## Major Update - 2026/7/18

### New Features

* Added local PVP mode, supporting two-player offline battles.

### Improvements

* Improved ball launch direction generation logic.
* Optimized ball spawn timing and spawn interval calculation.
* Refactored and optimized several Blueprint systems for better stability and maintainability.
